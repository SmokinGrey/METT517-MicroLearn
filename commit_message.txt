feat(notes): 학습 노트 기능 및 노트와 대화하기 기능 구현

'프로젝트 기반 학습' 기능의 핵심인 학습 노트(Learning Note) 아키텍처를 도입하고, 노트 전체와 대화할 수 있는 RAG 기반 채팅 기능을 구현합니다.

주요 변경 사항:
- **백엔드**:
  - `LearningNote` 및 `Source` 데이터베이스 모델을 도입하고 관계를 재설계.
  - `note_id`를 기반으로 동작하는 RAG 파이프라인(`rag_handler`)을 구현하여, 노트에 포함된 모든 소스를 대상으로 한 통합 검색 및 답변 생성 기능 제공.
  - 학습 노트 및 소스 관리를 위한 CRUD API 엔드포인트 추가.
  - 신규 로직에 맞춰 AI 자료 생성 및 소스 처리 로직을 업데이트.

- **프론트엔드**:
  - 사용자의 모든 학습 노트를 보여주는 `DashboardPage`를 구현.
  - 노트의 상세 정보(소스 목록)를 표시하고, 노트와 대화할 수 있는 `LearningNoteDetailPage`를 신규 생성.
  - `ChatInterface` 컴포넌트를 `noteId` 기반으로 동작하도록 수정하고, 상세 페이지에 통합.
